# 要求仕様書（RequirementsSpecification.md）

## 目的
未実装の機能のみを整理し、これからの開発対象を明確にする。

## 未実装の要件
### 1. ツールによる破壊ルール
- ツルハシ装備時のみ、土/草/石/砂/鉱石などを破壊できる
- 斧装備時のみ、木/葉などを破壊できる
- 対象外のブロックは破壊不可とする
- 影響範囲はプレイヤーの向きに依存し、右向き時は右側と下側のみを対象にする
- 影響範囲は3マスに縮小する（現在の4マスから変更）
- 直接接しているマスのみを対象とし、隣接以外への影響を禁止する

### 2. ハンマーの挙動
- 作業台の前で左クリックするとクラフトを実行する（当面はダミー処理で可）
- 収納箱/作業机などの設置物の前で右クリックすると所持アイテムに戻す

### 3. クラフトUIの拡張
- 作業机のオーバーレイ内にクラフトUIの土台を追加する
- レシピ/クラフト結果は後から差し替えられる構成にする

### 4. インタラクトのガイド表示
- プレイヤーが設置物に接触したときに「Eで開く」「左クリックでクラフト」などのガイドを表示する

## ブロック/設置物の一覧（要件整理）
### ブロック（blocks）
- 土: ツルハシで破壊可能
- 草: ツルハシで破壊可能
- 石: ツルハシで破壊可能
- 木: 斧で破壊可能
- 葉: 斧で破壊可能
- 砂: ツルハシで破壊可能
- 石炭鉱石: ツルハシで破壊可能
- 鉄鉱石: ツルハシで破壊可能
- 金鉱石: ツルハシで破壊可能
- ダイヤ鉱石: ツルハシで破壊可能
- レンガ: ツルハシで破壊可能

### 設置物（placeables）
- 木の壁: 破壊対象ではなく設置物扱い（通行可能、インタラクト対象）
- 木の扉（上側）: 破壊対象ではなく設置物扱い（通行可能、インタラクト対象）
- 木の扉（下側）: 破壊対象ではなく設置物扱い（通行可能、インタラクト対象）
- 梯子: 破壊対象ではなく設置物扱い（通行可能、インタラクト対象）
- 収納箱: ハンマー右クリックで回収、E/左クリックで開閉
- 作業机: ハンマー右クリックで回収、E/左クリックでクラフト

## 開発ステップ
1. ツール別の破壊判定と影響方向/範囲（3マス・接触のみ）を実装する
2. ハンマー右クリックで設置物を回収する処理を実装する
3. 作業机のクラフト処理（ダミー）とUI土台を追加する
4. 接触時の操作ガイド表示を追加する
