## パディングと描画方法
- 全スプライトシートは、各36x36スプライトの間に横方向パディングが入っています
- 現在のパディング幅: 6px
- 1コマの横幅は「36px + 6px」で計算し、参照位置は `srcX = index * (36 + 6)` を使います
- 画像の横幅は「(36 + 6) * 8 - 6 = 330px」になる（最後のコマの右にはパディングがない）
- 16コマのシート（8列 x 2行）の場合は、`col = index % 8`, `row = floor(index / 8)` で列と行を求め、`srcX = col * (36 + 6)`, `srcY = row * (36 + 6)` で参照位置を計算します
- 16コマの縦幅は「(36 + 6) * 2 - 6 = 78px」になる（行間は 6px）
- 参照がずれた時は「画像サイズ」「パディング幅」「srcX/srcY の計算式」が一致しているかを確認します
- 使用箇所: `js/render.js`（描画）, `js/ui.js`（アイコン表示）
